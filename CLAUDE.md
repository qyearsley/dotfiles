# Dotfiles Index

Personal configuration files for Unix-based systems.

## Structure

- `config-nvim/` - Neovim configuration (init.lua)
- `config-fish/` - Fish shell config and functions
- `zsh-functions/` - Zsh utility functions
- `claude-commands/` - Custom Claude Code slash commands
- `starship.toml` - Cross-shell prompt configuration
- `gitconfig` - Git configuration
- `zshrc` - Zsh configuration

See individual README files for details on each component.
